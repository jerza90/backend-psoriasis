# Configuration

## Application Properties

Key config in `src/main/resources/application.properties`:

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/psoriasis_db
spring.datasource.username=postgres
spring.datasource.password=postgres
spring.jpa.hibernate.ddl-auto=create-drop

stripe.secret-key=sk_test_...
stripe.webhook-secret=whsec_...

app.base-url=http://localhost:5173
```

## Database

Tables auto-generated by JPA:

- **users** — id, username, email, password_hash, full_name, enabled, otp_code, otp_expiry
- **orders** — id, email, full_name, product_type, product_id, amount, surcharge, total, currency, status, stripe_session_id, stripe_payment_intent

## Stripe Flow

1. Frontend calls `POST /api/checkout/create-session`
2. Backend creates Stripe Checkout Session, saves `Order` (status: `pending`)
3. Stripe redirects to `/thank-you?session_id=...`
4. Frontend verifies via `GET /api/checkout/session/{id}`
5. On `payment_status === "paid"`, download link serves PDF
6. Webhook (`POST /api/webhooks/stripe`) updates order status asynchronously
